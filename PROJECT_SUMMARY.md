# 太阳系 3D 可视化项目总结

> 本总结为本地文档，未提交到 GitHub。

## 一、项目概述

一个基于 Web 的交互式太阳系三维模型，支持**示意 / 真实 / 真比例**三种尺度模式，包含八大行星、卫星、彗星、矮行星、航天器、小行星带、柯伊伯带、奥尔特云及重要天象事件的可视化。项目以教育展示为目的，支持中英双语切换。

- **线上地址**：https://solarsystem.cc.cd
- **GitHub 仓库**：https://github.com/hahalei0517/solarsystem
- **部署平台**：Cloudflare Pages

## 三、已完成的主要功能

### 3.1 视觉与渲染

- **星空背景**：带大小、闪烁和光谱色温的恒星场，叠加银河尘埃层。
- **行星大气**：使用 `ShaderMaterial` 实现晨昏线暖色渐变的大气光晕，地球恢复柔和云层与夜光层。
- **Bloom 辉光**：提高阈值（0.62），避免背景恒星被糊掉，仅太阳/日冕产生辉光。
- **行星环系统**：土星卡西尼缝/恩克缝、天王星环、海王星环的程序化纹理。
- **太阳效果**：光球贴图、脉冲日冕、日珥弧。

### 3.2 外层太阳系

- **柯伊伯带**：三类群体
  - Plutinos（2:3 共振，~39 AU，红棕色）
  - 经典柯伊伯带天体（42–48 AU，蓝灰/红色）
  - 离散盘天体（50–90 AU，暗灰色，高偏心率）
- **奥尔特云**：球壳粒子分布，三种模式下位置经复核准确。
  - 示意：24–38.4 单位
  - 真实：180–288 AU
  - 真比例：~1.88M–3M 地球直径

### 3.3 航天器

包含 5 个深空探测器：旅行者 1/2、先驱者 10/11、新视野号。

- **示意/真实模式**：显示 3D 模型（抛物面天线 + RTG + 仪器吊杆 + 中心辉光），可点击，模型朝向飞行方向。
- **真比例模式**：显示环形定位标记 + 短粒子拖尾（36 天），按真实 AU 线性外推定位。

### 3.4 交互与 UI

- 三种时间速率：时 / 日 / 月 / 年
- 三种尺度模式：示意 / 真实 / 真比例
- 两种画质档位：高清 / 性能，含自适应降质
- 图层菜单：轨道、黄道面、标签、轨迹、小行星、奥尔特云、彗星、矮行星、航天器、天象事件、地轴、辉光、声音等
- 行星列表、信息卡、天象事件时间轴
- 键盘快捷键、移动端触摸操作、随光标旋转

### 3.5 教育内容

- 信息卡展示分类标签、质量、重力、表面温度、直径对比条。
- 天象事件：凌日、合、冲、大距、行星合相等几何关系可视化。
- 太阳黑子周期、北半球季节、彗星近日点倒计时、航天器与地球距离等。

### 3.6 部署与缓存

- `public/_headers` 配置：
  - `index.html`：`no-cache, must-revalidate`
  - `/assets/*`：`public, max-age=31536000, immutable`
- CSP 允许 Cloudflare Analytics beacon。

## 七、注意事项

- 示意模式为教育可视化做了距离/大小夸张，不是精密天文软件。
- 真实模式使用近似日心轨道元素，推荐在 1800–2050 年间查看。
- 真比例模式下外行星几乎不可见，用于直观感受太空的空旷。
