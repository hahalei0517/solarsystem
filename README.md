# 太阳系互动模型

一个面向科普和教学展示的太阳系运动模拟网页。当前主版本是 3D Three.js 版本，支持行星运动、卫星、轨道、示意/真实/真比例尺度、画质/性能模式、多颗知名彗星、重要天象标记、声音提示和辉光效果。

## 入口说明

- `index.html`：当前主入口，3D Three.js 太阳系模拟。
- `index-2d.html`：旧版/备用 2D Canvas 页面，保留用于对照和低配备用，不作为当前主开发入口。
- `serve.bat`：Windows 本地启动辅助脚本。引入 Vite 后，推荐优先使用 npm 命令启动。

## 科学精度边界

本项目用于教育可视化，不是精密天文软件或航天任务级星历工具。

- “示意”模式会夸张距离和大小，优先保证可读性。
- “真实”模式使用近似日心轨道元素展示八大行星位置，适合视觉近似，不适合精确观测预报。
- 代码中的行星近似星历基于 Standish/JPL 风格轨道元素，适用范围约为 1800–2050 年附近，越界后误差会增大。
- 行星半径、卫星轨道、月球/主要卫星位置、小行星带、柯伊伯带、彗星尾、辉光和声音均为视觉简化。
- 未模拟 n 体摄动、光行时、岁差章动、地理观测点、真实卫星星历等专业天文因素。
- 同样的精度边界会在网页内的“操作帮助”面板中提示，并在时间超出推荐范围时显示警告。

## 本地开发运行

首次运行先安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

然后在浏览器打开：

```text
http://localhost:5173/
```

如果 5173 端口被占用，请使用终端中 Vite 打印出的实际地址。

## 测试和构建

运行基础单元测试：

```bash
npm test
```

运行浏览器冒烟测试：

```bash
npm run test:browser
```

运行完整验证：

```bash
npm run test:all
```

构建静态产物：

```bash
npm run build
```

预览构建结果：

```bash
npm run preview
```

## 画质与性能模式

顶部“画质”控件可在“高清”和“性能”之间切换。

- 高清：保持当前视觉效果，包含辉光、较高像素比、阴影、完整星空和小行星带数量。
- 性能：关闭辉光，降低渲染像素比，禁用阴影，减少星空/小行星带绘制数量。
- 窄屏/移动端默认进入性能模式；桌面端默认进入高清模式。

## 尺度模式

顶部“尺度”控件可在三档之间切换：

- 示意：夸张距离和大小，优先保证可读性，是默认模式。
- 真实：使用近似日心轨道元素，1 场景单位 = 1 AU，展示八大行星真实相对轨道。
- 真比例：以 1 地球直径为 1 场景单位，行星、太阳、卫星按真实直径，轨道按真实 AU 显示。由于尺度悬殊（1 AU ≈ 1.17 万倍地球直径），外行星视野下行星几乎不可见，切换时镜头会自动拉远。本模式用于直观感受太空之空旷，存在轻微深度精度误差。

## 项目结构

```text
index.html                 3D 主页面
index-2d.html              旧版/备用 2D 页面
public/vendor/             构建时保留的 Three.js UMD 运行时文件
src/styles.css             页面样式
src/data/solar-system.js   太阳系数据、速度模式、近似星历
src/core/orbits.js         可测试的轨道和日期工具
src/core/events.js         可测试的天象扫描工具
src/app/state.js           模拟状态工厂
src/app/main.js            Three.js 场景和动画入口
src/app/ui/                信息卡、行星列表、控制栏模块
src/app/input-selection.js 3D 对象点击和悬停选择
tests/                     Vitest 单元测试和 Playwright 冒烟测试
textures/                  行星、月球、银河背景等贴图
```

## 资源说明

项目使用本地 Three.js UMD 文件和本地贴图资源，以保证离线/静态部署兼容性。构建需要的 UMD 运行时文件放在 `public/vendor/`，Vite 会将它们复制到 `dist/vendor/`。后续正式发布前，建议补齐每个贴图和第三方脚本的来源、版本和许可证说明。
